# Инструкция по работе с Markdown

## Выделение текста

* Для того чтобы сделать текст курсивом, необходимо поставить " * " до и после текста (* тектс *)
* Для того чтобы сделать текст жирным, необходимо поставит " ** " до и после текста (** текст **)
